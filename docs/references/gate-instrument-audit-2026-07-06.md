# 证伪关仪器对抗审计（2026-07-06，独立子agent）

结论摘要：信号与引擎层因果性干净（无前视泄漏）；但发现三个 critical——
C1 涨跌停判定在复权价上失效（一字板漏检→全网格系统性乐观）；
C2 随机对照未受市场过滤约束（MA60 beta 即可过关）；
C3 发现期检验被选择污染（840格挑极值后，同期对照/邻域检出力≈0），
真正防线只剩留出期，而原G2标准过软。
另 M1 ST 5%限价未建模、M2 留出期短样本剔除=幸存者偏差、M3 G4放水、
M4 统计功效不足（Calmar排序≈按噪音排序，应改按周聚类t）、m2 cxhzb权重倒置。

全部修复（2026-07-06 同日）：
- C1→比例判定+0.2%容差；M1→ST期(滚动40日|涨跌|≤5.5%)与退市前120日禁止入场
- C2/m4→置换对照改为"同过滤条件日抽样+按实际成交笔数配平"
- C3→G1/G3全部搬到留出期(2025-01~2026-07)执行，G1需≥99.4分位(8候选Bonferroni)
- M2→留出期不再剔除短样本；M3→G4改净利润口径+年度集中度≤85%
- M4→Stage-1主排序改按周聚类t统计量，候选门槛 n_trades≥800
- m2→cxhzb权重改为最近日最大

审计者总评（修复前）："纯噪音+MA60 beta 有现实概率通过四关；否定性结论可信，
PASS 结论不应被采信。" 修复后重跑全部网格。
